# Plus Check

Plus Check removes the uncertainty and anxiety of finding plus-size friendly seating anywhere.

## Structure

- `backend/`: Node.js + Express API (uses `sql.js` for cross-platform SQLite support)
- `frontend/`: React + Vite frontend

## Setup

### Backend
1. `cd backend`
2. `npm install`
3. `node init-db.js`
4. `node seed.js`
5. `npm start`

### Frontend
1. `cd frontend`
2. `npm install`
3. `npm run dev`
